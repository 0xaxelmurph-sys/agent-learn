# AI Agent Commerce Demo

Proyek ini adalah demo sederhana AI agent yang menangani transaksi otonom di e-commerce, terinspirasi dari Agentic Commerce (AI agents yang handle commerce mandiri, seperti di blockchain/DeFi).

## Fitur
- Simulate AI agent yang cek stok, proses pembayaran mock, dan konfirmasi transaksi.
- Menggunakan LangChain untuk agent framework dan OpenAI untuk decision-making.

## Cara Install dan Run
1. Clone repo: `git clone https://github.com/0xmukti/ai-agent-commerce-demo.git`
2. Install deps: `pip install -r requirements.txt`
3. Set OPENAI_API_KEY di environment.
4. Run: `python agent.py`
